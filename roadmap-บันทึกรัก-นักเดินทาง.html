Phase 1 — ทันที
Validate — ทดสอบตลาดก่อนลงทุน
เอา 2 ไฟล์ที่มีอยู่ไปโชว์คน 20–30 คน ถามว่าจะใช้จริงไหม จะจ่ายเงินไหม ไม่ต้องเขียนโค้ดเพิ่มแม้แต่บรรทัดเดียว
⏱ 1–2 สัปดาห์ · ค่าใช้จ่าย ฿0
🌐
Phase 2 — 2–4 เดือน
Web App MVP — เว็บใช้งานได้จริง
สร้าง Next.js app เชื่อม Supabase (database+auth) เชื่อม Claude API (AI สรุป) เชื่อม Mapbox (แผนที่) Deploy บน Vercel ใช้งานได้ผ่าน browser มือถือทันที ยังไม่ต้องเป็นแอพ Store
⏱ 2–4 เดือน · ค่าเริ่มต้น ฿0–3,000/เดือน
📱
Phase 3 — เมื่อมี user แล้ว
Mobile App — แอพใน App Store + Google Play
แปลง Web App เป็น React Native (Expo) ให้ได้ทั้ง iOS + Android จากโค้ดชุดเดียว Submit เข้า App Store และ Google Play ได้เลย
⏱ 2–3 เดือนเพิ่มเติม · ค่า Apple $99/ปี + Google $25 ครั้งเดียว
🏢
Phase 4 — ระยะยาว
Full Platform — Creator monetization + Video streaming
Video (Mux) · Payment (Stripe/Omise) · Creator revenue share · Family Space · AI Recommendation engine · Admin dashboard ครบ — เฟสนี้ต้องการทีมและงบลงทุนจริงจัง
⏱ 6–12 เดือน · งบ ฿500K–2M ขึ้นอยู่กับทีม

▸ Phase 2 — Backend + AI + Maps
⚡
Supabase
Database + Auth + Storage (แทน Firebase ที่ดีกว่า)
ฟรี 500MB
สมัครที่ supabase.com · Free tier ใช้ได้จริงสำหรับ MVP
ได้อะไร: PostgreSQL database · User Auth (email/Google/Apple login) · File Storage สำหรับรูปและวิดีโอ · Real-time subscriptions
จำกัด free: 500MB storage · 2 projects · เกินจ่าย $25/เดือน
🤖
Anthropic API (Claude)
AI สรุปทริป + ตามรอย + แนะนำ
Pay per use
สมัครที่ console.anthropic.com · จ่ายตามที่ใช้จริง
ราคา Claude Sonnet 4.5: Input $3/1M tokens · Output $15/1M tokens
ประมาณค่าใช้จ่าย: สรุปทริป 1 ครั้ง ≈ ฿0.15–0.50 · 1,000 user/วัน ≈ ฿150–500/วัน
วิธีเริ่ม: สร้าง API key → ใส่ใน .env ของ Next.js → เรียกผ่าน fetch
🗺️
Mapbox
Interactive World Map
ฟรี 50K loads/เดือน
สมัครที่ mapbox.com · Free tier 50,000 map loads/เดือน
ใช้ทำอะไร: แผนที่โลก Interactive · ปักหมุดสถานที่ · Clustering หมุดหลายอัน · 3D terrain
เกิน free: $0.50 ต่อ 1,000 loads ถัดไป
☁️
Cloudflare
CDN + ป้องกัน DDoS + R2 Storage
ฟรี
สมัครที่ cloudflare.com · Free plan เพียงพอสำหรับ MVP
ใช้ทำอะไร: เชื่อม domain ของตัวเอง · CDN เร็วขึ้นทั่วโลก · R2 Storage เก็บรูป/วิดีโอราคาถูกกว่า AWS S3

🌐 PHASE 2 — Web App MVP (2–4 เดือน)
3
ติดตั้ง Tools ที่จำเป็น
ลงโปรแกรมเหล่านี้บน Mac/Windows
Node.js — nodejs.org (ดาวน์โหลดฟรี)
VS Code — code.visualstudio.com (code editor ฟรี)
Git — มากับ Mac หรือดาวน์โหลดที่ git-scm.com
4
สร้าง Next.js Project
เปิด Terminal แล้วพิมพ์
npx create-next-app@latest banthuek-rak
เลือก: TypeScript ✓ · Tailwind ✓ · App Router ✓
แล้วใช้ HTML ไฟล์เดิมเป็น reference แปลงเป็น React components ทีละหน้า
5
เชื่อม Supabase (Database + Login)
1. สมัคร supabase.com → New Project
2. สร้าง Tables: users · posts · pins · families
3. เปิด Auth → เลือก Email, Google, Apple login
4. ติดตั้ง: npm install @supabase/supabase-js
5. ใส่ URL + API key ใน .env.local
→ supabase.com
6
เชื่อม Claude API (AI สรุปทริป)
1. สมัคร console.anthropic.com → Create API Key
2. ใส่ใน .env.local: ANTHROPIC_API_KEY=sk-ant-xxx
3. สร้าง API Route: /app/api/summarize/route.ts
4. เรียก Claude ด้วย prompt เช่น "สรุปทริปนี้เป็นภาษาไทย: [โพสต์ของ user]"
5. แสดงผลใน AI Summary Card ที่ออกแบบไว้แล้ว
→ console.anthropic.com
7
เชื่อม Mapbox (แผนที่โลก)
1. สมัคร mapbox.com → Copy Access Token
2. npm install mapbox-gl react-map-gl
3. แทนที่ SVG map ใน prototype ด้วย Mapbox GL JS map จริง
4. ดึง pins จาก Supabase มาแสดงเป็น markers บนแผนที่
→ mapbox.com
8
Deploy บน Vercel
1. Push โค้ดขึ้น GitHub
2. สมัคร vercel.com → Import Project
3. ใส่ Environment Variables (.env ทั้งหมด) ใน Vercel dashboard
4. กด Deploy → ได้ URL ทันที เช่น banthuek-rak.vercel.app
5. ซื้อ Domain เช่น banthuekrak.com (฿400–800/ปี) → ต่อใน Vercel
→ vercel.com

สมัคร Apple Developer + Google Play
Apple: developer.apple.com/programs → $99/ปี → ใช้บัตรเครดิต

11
Build + Submit แอพเข้า Store
ใช้ EAS Build (Expo Application Services) build แอพบน cloud
eas build --platform ios → ได้ .ipa file → Upload ใน App Store Connect
eas build --platform android → ได้ .aab file → Upload ใน Google Play Console
กรอก metadata: ชื่อแอพ · คำอธิบาย · Screenshots · Privacy Policy
Privacy Policy: ต้องมี → ใช้ termly.io หรือ privacypolicygenerator.info ฟรี
