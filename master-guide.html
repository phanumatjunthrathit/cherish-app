🚀 PHASE 1 — ทำเลยวันนี้ ฟรี
1
สมัคร GitHub · github.com
กด Sign up → ใส่ email + password → ยืนยัน email → เสร็จสิ้น
ใช้เวลา: 3 นาที
→ github.com
2
สร้าง Repository ใหม่
กด + มุมบนขวา → New repository → ชื่อ banthuek-rak → เลือก Public → กด Create repository
3
เปลี่ยนชื่อและอัปโหลดไฟล์
สำคัญมาก! เปลี่ยนชื่อ banthuek-rak-preview.html → index.html ก่อน
แล้วกด uploading an existing file → ลาก drop ไฟล์ทั้ง 4 ไฟล์ → กด Commit changes
4
เปิด GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Save → รอ 2 นาที
5
ได้ URL → ส่งให้คนทดสอบ
URL รูปแบบ yourname.github.io/banthuek-rak
ส่งให้คน 20–30 คน ถามว่า: "ถ้าแอพนี้มีจริงจะใช้ไหม? จ่ายเดือนละเท่าไหร่?"
ถ้า 15/20 บอกใช่ → เดินหน้า Phase 2 ได้เลย
🌐 PHASE 2 — Web App (2–4 เดือน)
6
ติดตั้งเครื่องมือ
Node.js → nodejs.org ดาวน์โหลดฟรี
VS Code → code.visualstudio.com ฟรี
Git → มากับ Mac หรือ git-scm.com
7
สร้าง Next.js Project
รัน: npx create-next-app@latest banthuek-rak
เลือก: TypeScript ✓ · Tailwind ✓ · App Router ✓
แล้วให้ Developer ใช้ไฟล์ HTML เป็น design reference แปลงเป็น React components
8
สมัคร Supabase → เชื่อม Database + Login
1. สมัคร supabase.com → New Project
2. สร้าง Tables: users · posts · pins · families · media
3. เปิด Auth → เปิด Google + Apple Login
4. npm install @supabase/supabase-js
5. ใส่ URL + Key ใน .env.local
→ supabase.com
9
สมัคร Anthropic API → เชื่อม AI
1. สมัคร console.anthropic.com → Create API Key
2. ใส่ใน .env.local: ANTHROPIC_API_KEY=sk-ant-xxx
3. สร้าง API Route: /app/api/ai/route.ts
4. ฟีเจอร์ที่ใช้: สรุปทริป · วางแผนตามรอย · เขียนคำบรรยาย · AI Chat ตอบคำถาม
→ console.anthropic.com
10
สมัคร Mapbox → เชื่อมแผนที่
1. สมัคร mapbox.com → Copy Access Token
2. npm install mapbox-gl react-map-gl
3. แทนที่ SVG map ด้วย Mapbox GL JS จริง
4. ดึง pins จาก Supabase แสดงบนแผนที่
→ mapbox.com
11
Deploy บน Vercel → เว็บออนไลน์
1. Push โค้ดขึ้น GitHub
2. สมัคร vercel.com → Import Project
3. ใส่ Environment Variables ทั้งหมดใน Vercel Dashboard
4. กด Deploy → ได้ URL ทันที เช่น banthuek-rak.vercel.app
5. ซื้อ Domain banthuekrak.com (฿400–800/ปี) → ต่อใน Vercel
→ vercel.com
📱 PHASE 3 — Mobile App (2–3 เดือนเพิ่ม)
12
ติดตั้ง Expo + สร้าง Mobile App
npm install -g expo-cli
npx create-expo-app banthuek-rak-app
ทดสอบบนมือถือผ่านแอพ Expo Go (ฟรี ไม่ต้องขึ้น Store)
→ expo.dev
13
สมัคร Apple Developer Program
ไปที่ developer.apple.com/programs → กด Enroll → $99/ปี → จ่ายบัตรเครดิต
ต้องการ: Mac หรือ MacBook · Apple ID · รอ 1–2 วันทำการ
14
สมัคร Google Play Console
ไปที่ play.google.com/console → Create account → $25 ครั้งเดียว → จ่ายบัตรเครดิต
ใช้ได้: Mac หรือ Windows · Gmail account · อนุมัติเกือบทันที
15
Build แอพด้วย EAS + อัปโหลด
รัน: eas build --platform ios → ได้ .ipa → อัปผ่าน App Store Connect
รัน: eas build --platform android → ได้ .aab → อัปผ่าน Google Play Console
เตรียมเพิ่ม: App Icon · Screenshots · Privacy Policy URL (termly.io ฟรี) · คำอธิบายแอพภาษาไทยและอังกฤษ
16
รอ Review → แอพออนไลน์ 🎉
Apple: รอ 1–3 วัน → อนุมัติ → ขึ้น App Store
Google: รอ 1–7 วัน → อนุมัติ → ขึ้น Play Store
ผู้ใช้สามารถดาวน์โหลดแอพได้แล้ว!
